에이전트 시대의 개막 선언

2026년 6월 14일~20일 · 앱이 물러나고 에이전트가 온다

이번 주는 Microsoft가 Build 2026에서 Windows를 AI 에이전트 플랫폼으로 공식 선언하면서 업계 전체가 들썩였다. 모델 출시는 그 어느 때보다 빠르고, 중국 모델들은 서방 플래그십과 사실상 동급에 올랐으며, 콜로라도는 AI 규제를 반 년 늦추기로 했다. 조용한 주는 아니었다.

이주의 메인 스토리

Microsoft Build 2026: Windows를 에이전트 OS로 선언하다

6월 2~3일 샌프란시스코에서 열린 Build 2026에서 Satya Nadella는 Windows를 AI 에이전트를 실행하는 플랫폼으로 재정의했다. Agent Framework(MAF) 1.0 GA, 7종의 MAI 자체 모델(추론·코딩·이미지·음성 등), Project Solara(에이전트 전용 Android 기반 기기 플랫폼), Microsoft Scout(항상 켜져 있는 자율 에이전트) 등이 한꺼번에 쏟아졌다.

특히 눈에 띄는 건 Web IQ다. 모델 불가지론적(model-agnostic)이고 MCP 네이티브로 설계된 AI 우선 웹 검색 스택으로, 에이전트가 실시간으로 웹을 읽고 행동하는 기반이 된다. 앱을 열고 닫는 컴퓨팅에서 에이전트가 알아서 작업을 처리하는 컴퓨팅으로의 전환이 공식화된 셈이다.

🤖 모델 전쟁: 6월이 역대 최대 출시 달이 되다

  • Gemini 3.5 Pro, 기업 고객 한정 프리뷰 개시 ↗
    200만 토큰 컨텍스트와 'Deep Think' 추론 모드를 탑재하고 Google I/O에서 깜짝 발표됐지만, 일반 출시는 아직이다. Sundar Pichai가 6월 내 출시를 약속한 만큼 마지막 2주가 관건.
  • MiniMax 3, Elo 1528로 서방 플래그십과 동급 — 토큰당 가격은 $0.53 ↗
    중국 상하이 스타트업 MiniMax가 내놓은 MiniMax 3는 성능 면에서 GPT-5 급이면서 가격은 1/5 수준. 미·중 AI 격차가 사실상 사라졌다는 평가가 나온다.
  • DeepSeek V4, MIT 오픈웨이트로 1.6조 파라미터 공개 ↗
    V4-Pro(1.6T, 활성 49B)와 V4-Flash(284B, 활성 13B) 두 버전을 MIT 라이선스로 공개. $0.87/M 출력 토큰으로, 동급 성능 클로즈드 모델 대비 비용이 획기적으로 낮다.

🛠️ 에이전트 인프라가 Production 급으로

  • Microsoft Agent Framework 1.0 GA — AutoGen + Semantic Kernel 통합 완료 ↗
    .NET과 Python을 동일한 API로 지원하는 오픈소스 에이전트 SDK. 이제 "멀티에이전트"는 실험이 아니라 프로덕션 선택지가 됐다.
  • Google, ADK 1.0 Java·Go 버전 출시 ↗
    Python에 이어 Java와 Go까지 지원 확대. 엔터프라이즈 백엔드에서 에이전트를 직접 심을 수 있게 됐다는 의미다.
  • Anthropic Claude Agent SDK, 6월 15일부터 별도 크레딧 과금 시작 ↗
    에이전트 SDK가 '베타 무료'에서 벗어나 본격적인 상업화 단계에 진입. 에이전트 인프라 전반이 성숙 단계로 접어든 신호.

💰 산업 AI로 흐르는 큰돈

  • PhysicsX, 3,900억 원(약 $300M) Series C 조달 — 밸류에이션 약 3.1조 원 ↗
    Temasek 주도, NVIDIA·Siemens 참여. 항공우주·반도체·방위산업 분야 엔지니어링 설계에 물리 AI를 접목. F1 엔지니어 출신 창업자가 만든 회사가 기업 가치 3조 원을 돌파했다는 게 상징적이다.
  • PointFive, $60M Series B — "AI로 클라우드 비용을 줄여드립니다" ↗
    Accel 주도. AI가 클라우드 비용을 절감하는 아이러니한 구도지만, 기업 입장에서는 ROI가 명확해 투자자들이 좋아하는 유형이다.
  • 2026년 VC 전체의 33%가 AI로 — Series A 평균 투자액 $51.9M ↗
    시장 전체로 돈은 흘러들어오지만, 실제로 받는 곳은 소수다. "고객, 리텐션, 반복 가능한 영업" 없이는 Series A도 어렵다는 게 현실.

⚖️ 미국 AI 규제 — 후퇴인가, 재조정인가

  • 콜로라도 AI법 개정: 시행 6개월 연기, 의무 대폭 축소 ↗
    원래 6월 30일 시행 예정이었던 콜로라도 AI법이 SB 189로 개정되어 2027년 1월 1일로 연기됐다. 백악관 압력과 산업계 로비가 작용한 결과. "EU 모델에서 멀어지는 미국"이라는 평가가 나온다.
  • 캘리포니아 Frontier AI 투명성법은 건재 — 위반 시 최대 10억 원 벌금 ↗
    연매출 5억 달러 이상 프론티어 모델 개발사는 리스크 공개·사고 보고·내부 고발자 보호 의무를 진다. 연방이 침묵하는 동안 주들이 규제를 채워가고 있다.

🔬 연구: 아키텍처는 이제 스케일링 너머를 본다

  • HippoRAG 2, 연상 기억 성능 대폭 개선 ↗
    RAG(검색 증강 생성)의 고질적 약점인 "사실 + 연상 기억" 처리를 크게 향상시킨 논문. 파편화된 정보를 연결하는 능력이 에이전트 응용에서 특히 중요해진다.
  • 2026년 연구 트렌드: 하이브리드 아키텍처·표현 기하학으로 이동 ↗
    "더 크게"만이 답이 아니라는 공감대가 연구 커뮤니티에 형성됐다. Transformer 너머의 아키텍처와 내부 표현(representation) 해석 연구가 올해 두드러진다.
